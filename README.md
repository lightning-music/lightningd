lightningd is the server component of lightning.



# Build

Prerequisites:

- [liblightning](https://github.com/lightning/liblightning)
- [lightning/go](https://github.com/lightning/go)



# Install

```shell
$ go get github.com/lightning/lightningd
```

### Usage

```shell
$ lightningd -h
```
